  • node express ejs passport Bootstrap Flat-UI

学习Node

的Node学习之旅

第一个任务

今天是正式学习node的并准备制作的第一个实例 (一个登登陆页面) 。

准备 

PC一台(*穷屌丝只能用windows10*)
Modules：express、ejs、mongoose、passport、passport-local、passport-local-mongoose
DB：mongodb
UI：Flat-UI(*a Free Bootstrap Framework and Theme*)

开始

Node安装等不再讲述

初始化

准备好需要的modules

1. 安装express并初始化

npm install express -g

由于安装的express为v4.*.* 因此还需要安装express-generator来使用exoress命令。

npm install express-generator -g

安装完成之后，执行

express -e Login(的项目名称)

会在Login目录下新建出如图所示express骨架结构和ejs模板

然后将Login中将package.json中的包安装到Login目录中去,运行

npm install

安装ejs-mate 来使用母版页

npm install ejs-mate --save

在view目录下新建一个母版页boilerplate.ejs

要使页面使用母版页只要在ejs头上中增加<% layout('boilerplate') -%>

修改app.js

  1. 增加 var engine=require('ejs-mate');
  2. 增加 app.engine('ejs', engine);
  3. 在app.js最后加上

app.listen(8000,function(){console.log('监听8000端口')});

至此，网站就可以运行了:
node app.js
可以通过浏览器访问http://localhost:8000

安装passport、passport-local、mongoose、passport-local-mongoose并将他们写入package.json
